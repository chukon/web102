JS 101

Overview

This course gives an overview of JavaScript and programming generally. It does not cover libraries like jQuery. The course is divided into 3 lessons (~2hours each):

  • Lesson 1: Variables & Functions
  • Lesson 2: Control Flow & Arrays
  • Lesson 3: Objects

Prerequisite

The students should have some familiarity with HTML/CSS (classes/ids). They do not need prior programming experience.
